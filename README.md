Программа предназначена для контроллера Arduino Uno и предназначена для создания системы для интеллектуальных игр, где важна скорость ответа.

Для работы нужно 5 кнопок и 5 цифровых пинов под эти кнопки, 4 цифровых пина под выходы для индикаторов (свет или звук), которые надо указать в начале программы (17-18 строки кода).
Одна из кнопок — мастер-кнопка, означает старт и стоп-раунда (после стопа время и нажатия сбрасываются). Одно нажатие после старта программы означает «старт», каждое следующее нажатие меняет состояние программы.


4 кнопки для команд, после нажатия «старт» может быть нажата только одна кнопка, индикатор загорается, остальные кнопки блокируются. После команды «Стоп» любое нажатие кнопки будет вызывать мигание индикатора.


Возможно также подключение дисплея через 2 аналоговых дисплея и вывод на него доп. информации (функция пока не настроена и не подключена в программу).


После нажатия «старт» можно задать время на ответ и на каких промежутках будет инидикация оставшегося времени (настроки таймера: 77-80)
