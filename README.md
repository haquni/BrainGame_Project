Программа предназначена для контролера ArduinoUno и предназначена для создания системы для интелектальных игр ,где важна скорость ответа.


Для работы нужно 5 кнопки и 5 цифровых пина под эти кнопки ,4 цифровых пина под выходы для индикаторов(Свет или Звук)которые надо указать в начале программы.

Одна из кнопок - мастер кнопка означает старт и стоп раунда(после стопа время и нажатия сбрасываются) Одно нажатия после старата программы означает "Старт" каждое следущие нажатие меняет состояние программы


4 кнопки для команд после нажатия "Старт" может быть нажата только одна конпка ,инидикатор загорается остальные кнопки блокируются,
После команлы "Стоп" любое нажатие кнопки будет вызывать мигание индикатора

Возможно так же подключение дисплея через 2 аналоговых дисплея и вывод на него доп информации (функция пока не настроена и не подключена в программу)

После нажатия "Старт" можно задать время на ответ и на каких промежутках будет инидикация оставшегося времемни
